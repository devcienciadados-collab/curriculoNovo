# cvlaunch — Editor de Currículos

Link: https://abre.ai/cvlaunch

O cvlaunch é um criador e editor de currículos interativo para apresentar sua carreira com profissionalismo. Crie um currículo que conta sua história, destaca suas habilidades e te diferencia no mercado de trabalho.

## Funcionalidades

- Autenticação: login e registro com Google ou GitHub
- Editor em tempo real para criação e personalização do currículo
- Armazenamento no banco de dados dos currículos criados
- Geração de conteúdo com Inteligência Artificial (sugestões e otimizações)
- Funcionalidades premium com compra via Stripe
- Exportação para PDF a partir de componentes HTML, preservando o layout

## Tecnologias

- Next.js 14 — Framework React para aplicações modernas
- TypeScript — Tipagem estática para maior confiabilidade
- Auth.js (NextAuth) — Autenticação completa
- shadcn/ui — Componentes de interface reutilizáveis
- Tailwind CSS — Estilização rápida e responsiva
- Drizzle ORM — Gerenciamento simples e eficiente do banco de dados
- Puppeteer — Automação de navegador (geração de PDFs)
- Stripe — Pagamentos e monetização
